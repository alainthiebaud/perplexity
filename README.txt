
Chat149 — Lancement & Dépannage rapide
--------------------------------------
1) Double-cliquez RUN.bat (ou: python main.py)
2) Onglet "Fichiers & Config" -> cliquez "Vérifier les chemins".
   - Corrigez les chemins si nécessaire (les changements sont mémorisés).
3) "Charger les données" -> puis "Préparer Répartition".
4) Onglet "Sorties": générez XLSX/PDF, "Rafraîchir la liste" puis double-cliquez un fichier pour l’ouvrir.

Dépendances (si requis):
  pip install openpyxl reportlab
